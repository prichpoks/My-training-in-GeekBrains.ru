Задание 1
Практикоовал переиещение по директориям.
Вернутся в домашнюю директори можно с помощью команды cd ~; cd; 
Перемещение на уровень выше cd "Имя директории"
Перемещение на уровень ниже cd ..
Можно переместиться в конкретную директорию по обсалютному пути cd /usr/local/bin/

Задание 2
ls - список файлов в директории.
	ключи:
		l - показать подробный список файлов с их пораметрами
		a - показать скрытые файлы
rm - удалить файл.
	ключи:
		r - удалить директорию рекусивно со всеми вложениями
rmdir - удалить пустую директорию 
mv - переместить/переиминовать файл
cp - копировать файл
	ключи:
		r - копировать директорию рекурсивно со всеми вложенными файлами
mkdir - содать директоию
touch - создать пустой файл
cat "name" - отоброзить содержимое файла в терминале
vim "name" - открыть новый файл или существующий в редакторе vim
